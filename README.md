# Wifi-capture-

Automatically grab WiFi info from any computer in milliseconds
⚡ How it works: Extracts saved WiFi passwords and network details that are already stored on the target machine.
⚠️ Important: Only works on WiFi networks the computer has connected to before. No saved network = no data.


---> change this : > wifiName = "yourwifi" 
        to your wifi name.

#HOW TO USE 
run this on the device that already connected to the wifi before and the information will be saved in apple.txt 
OR you can use this on usb flash drive to auto run it when insert the usb
